O que são tags semânticas?

Elementos semânticos equivalem a elementos com significado. Os elementos <div> e <span> são exemplos de tags que não tem significado semântico nenhum. Já <form>, <table> e <article> são exemplos de tags semânticas com significado.

Elementos semânticos em HTML

Os elementos semânticos podem dar significados a tags que não tem significado, exemplo: <div id="nav">, <div class="header" e <div id="footer"> Com elementos semânticos podemos usar para definir diferentes partes de um site web:

Tag <main> - Definição: O conteúdo dentro da <main> deve ser exclusivo do documento. Não deve conter nenhum conteúdo repetido em documentos, como barras laterais, links de navegação, informações de direitos autorais, logotipos de sites e formulários de pesquisa. Não deve ter mais de um <main> em um documento.

O <main> não deve ser descendente de um elemento <articls>, <aside>, <footer>, <header> ou <nav>.

A tag <strong> é usada para definir textos de grande importância. O conteúdo interno normalmente é exibido em negrito. Use a tag <b> para especificar texto em negrito sem qualquer importância extra!

A <del> tag define o texto que foi excluído de um documento. Os navegadores geralmente traçam uma linha no texto excluído. A tag <ins> define um texto que foi inserido em um documento. Os navegadores geralmente sublinham o texto inserido. A tag <ins> é usada para definir o texto enfatizado. O conteúdo interno normalmente é exibido em itálico .

A <sub> tag define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O

A tag <sup> define texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé, como WWW[1]
